animaux que j'aime animaux que je n'aime pas
les ours les araignés
les éléphants les souris
les chevaux les serpents
les singes les requins
les chats les rats
les baleines les scorpion

pourquoi je ne les aimes pas:

  1. les araignés: elles sont moche
  2. les serpent: j'aime pas leurs tête
  3. les requins: ils peuvent me manger
  4. les rats: par ce qu'ils sont moche
  5. les scorpion: par que peuvent me tuer
